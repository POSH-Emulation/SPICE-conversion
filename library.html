Topology library

Language

The topology library is a set of SPICE-like sub-circuits that are kept as a reference to known cells and their functions. See the cells at $DEGEM_ROOT/topologies/*.sp The format of the netlists is based on SPICE, with the following super-set features:

  • A circuit name can be repeated, provided it contains the pound symbol (#), which gets internally substituted with a running index. e.g. .subckt amplifier# in out vdd gnd

  • An instance name can also be repeated with the pound symbol, provided they keep the SPICE card convention. e.g. m# drain gate gnd gnd nmos

  • The names used in the circuit have meaning:

    • A library cell’s name, up until a suffix number (or pound) gives the matched cell its function. e.g. a design cell amp_out is matched to amp_stage#, which implies to the tool that it is a single input amplifier.

    • Interface nets can only be named in the following order: (analog|digital|bias|ctrl|osc)_(in|out)\N+ . Net names do not support the auto-numbered pound notation.

    • Transistors are listed with technology-neutral types: nmos, pmos, npn and pnp.

    • No values are assigned to transistor parameters or component sizes.

    • Supply nets must be named vdd and gnd.

    • Internal nets’ names have no restrictions.

  • A cell can be autoamtically P/N mirrored to include both cell and its reflection in the library by adding a pragma comment: *> npsym . Since mirroring re-enters a cell into the library, it must be named using the pound notation.

Adding a Design Cell to the Library

Once design sub-circuit was identified as a new type of cell, it can be added to the library be recognized automatically in the future. Use the following steps to prepare a library cell:

  1. Copy the .subckt ... .ends block into one of the appropriate files in the library or create a new one.

  2. Rename the sub-circuit according to its function.

  3. Identify and search/replace the interface nets’ names to reflect their function and directionality.

  4. Identify and search/replace supplies with vdd and gnd

  5. Replace all transistor types with nmos, pmos, npn and pnp.

  6. Remove all parameters assignments

  7. Delete dummy transistors (source-drain short, source-gate short etc.)

re-Hashing the Topology Library

DEGEM’s ability to match design sub-circuits with library cells depends on it having an up-to-date table of canonical labels. Once cells were added to the library, they must be compiled so following DEGEM runs can match to them.

To re-compile the library, run: degem -ct &
